## [1.0.0] - 2026-03-03

### Added
- Initial release of CBK CORF Assessment Toolkit
- Complete framework overview with 876 total controls across 27 domains
- Three baseline explorers:
  - Cyber Resilience Baseline (519 controls, 6 domains)
  - Operational Resilience Baseline (146 controls, 8 domains)
  - Third-Party Risk Management Baseline (211 controls, 13 domains)
- Interactive self-assessment questionnaires (10 questions per baseline)
- Automated maturity scoring system (Levels 1-5)
- Personalized recommendations based on assessment results
- Bilingual support (English/Arabic) with RTL layout
- Responsive design for mobile, tablet, and desktop
- Resources hub with links to official CBK documentation
- Educational disclaimer and compliance guidance
- Professional gradient design system
- Smooth animations and transitions
- Institution type selection (5 types)
- Detailed domain breakdown for each baseline
- Maturity level visualization with progress bars
- Score circle with color-coded results
- Reset functionality for assessments

### Design Features
- Custom CSS variable-based design system
- Blue gradient color scheme matching CBK branding
- Card-based layout with hover effects
- Accessible form controls and navigation
- Mobile-first responsive breakpoints
- Professional typography and spacing

### Localization
- Complete English translation
- Complete Arabic translation
- Seamless language toggle functionality
- RTL layout support for Arabic
- Localized content for Kuwait financial sector

### Documentation
- Comprehensive README.md
- MIT License
- Contributing guidelines
- Security policy
- Code of conduct
- Issue templates

- Firefox 88+

- Edge 90+
