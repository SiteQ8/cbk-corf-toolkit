> **Educational Web-Based Tool for Central Bank of Kuwait Cyber and Operational Resilience Framework (CORF) Compliance Assessment**

## 🎯 Overview

The **CBK CORF Assessment Toolkit** is a comprehensive, bilingual (English/Arabic) web application designed to help financial institutions in Kuwait understand, assess, and prepare for compliance with the Central Bank of Kuwait's **Cyber and Operational Resilience Framework (CORF)** issued in December 2025.

This tool provides:
- **Framework Overview**: Complete breakdown of CORF structure and requirements
- **Baseline Explorer**: Detailed information on all three baselines (Cyber, Operational, TPRM)
- **Self-Assessment**: Quick maturity evaluation across all baseline domains
- **Resources Hub**: Links to official documentation and guidance materials

## 🚀 Features

### 📊 Three Comprehensive Baselines

1. **Cyber Resilience Baseline** - 519 controls across 6 domains
2. **Operational Resilience Baseline** - 146 controls across 8 domains  
3. **Third-Party Risk Management Baseline** - 211 controls across 13 domains

- **English** and **Arabic** interface

## 📥 Installation

### Option 1: Direct Usage (Recommended)

1. Download `index.html`
2. Open in any modern web browser
3. No server or dependencies required

### Option 2: Local Web Server

```bash
# Clone the repository
git clone https://github.com/siteq8/cbk-corf-toolkit.git

# Navigate to directory
cd cbk-corf-toolkit

# Option A: Python 3
python -m http.server 8000

# Option B: Node.js (if you have http-server installed)
npx http-server

# Option C: PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## 📖 Usage Guide

### 1. Framework Overview
- Navigate to the **Overview** tab
- Review total controls, domains, and baselines
- Understand key framework features and requirements

### 2. Explore Baselines
- Click the **Baselines** tab
- Select a baseline card to view detailed domain breakdown
- Review control counts and structure

### 3. Conduct Self-Assessment
- Go to **Self-Assessment** tab
- Select your institution type (Bank, Finance Company, etc.)
- Choose baseline to assess
- Answer the compliance questions
- Click **Calculate Maturity Score** for results

### 4. View Results
- Review your maturity level (1-5)
- Read personalized recommendations
- Plan next steps for compliance improvement

### 5. Access Resources
- Visit **Resources** tab
- Access official CBK CORF documentation
- Explore related regulatory guidance

## 📋 CORF Framework Summary

### Framework Scope
- **Total Controls**: 876
- **Total Domains**: 27
- **Baselines**: 3
- **Maturity Levels**: 5 (Initial → Ad-hoc → Baseline → Advanced → Innovative)

### Key Requirements
- Board-level accountability for cyber and operational resilience
- Risk-based supervisory tiering
- Continuous monitoring and improvement
- Third-party risk management
- Incident response and crisis management
- Regular testing and validation
